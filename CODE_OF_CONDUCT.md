# Code of Conduct

Token Saver Loop is a small open-source project for people experimenting with AI-assisted development workflows. The community should be practical, respectful, and welcoming to users with different tools, models, languages, and experience levels.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially when discussing model behavior or workflow preferences.
- Keep criticism focused on ideas, evidence, code, docs, and user experience.
- Be clear when something is opinion, benchmark, anecdote, or verified result.
- Help keep issues and discussions useful for future readers.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Dismissive comments about a person or group rather than the work.
- Publishing private information without permission.
- Spam, unrelated promotion, or repeated off-topic posting.
- Deliberately misleading claims about security, cost savings, benchmarks, or project behavior.

## Reporting

If you see behavior that makes the project unsafe or unwelcoming, open a GitHub issue if public discussion is appropriate. For private reports, contact the repository owner through the GitHub profile associated with this repository.

Reports should include:

- what happened
- where it happened
- who was involved
- any relevant links or screenshots

Maintainers may remove content, lock conversations, or block users when needed to keep the project usable and respectful.

## Scope

This code of conduct applies to this repository, its issues, pull requests, discussions, and project-related community spaces controlled by the maintainers.
